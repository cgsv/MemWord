Memword

An application for memorizing English words using C#.

Welcome to Project MemWord.

This is a simple application for memorizing word on your Windows computer.

I wrote it because I need a light-weighted program to enlarge my English vocabulary, and I could not find a good one.

It is written in C#, and using an English word database with about 150,000 words and their meanings. If you have Lingoes Dictionary voice package installed, you could add the voice file path and the application will automatically pronounce each word as you watch it. Also, there is a text area for you to type the word as you memorize it.
